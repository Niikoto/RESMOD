## Sequência de Fibonacci e Estimativas Ágeis

<p>A sequência de Fibonacci, descrita por Leonardo de Pisa (Fibonacci), é uma sucessão infinita de números inteiros em que cada termo, a partir do terceiro, é a soma dos dois termos anteriores.</p>
<p><i>Exemplo da sequência:</i></p>

##### <i> 0, 1, 1, 2, 3, 5, 8, 13, 21, ...</i>

<p> No contexto de metodologias ágeis, como Scrum, essa sequência é amplamente utilizada para estimar esforço e complexidade de tarefas por meio de <b>Story Points.</b></p>
<p>Para facilitar o uso prático é comum utilizar uma versão adaptada da sequência: </p>

##### <i>0, 1, 2, 3, 5, 8, 13, 20, 40, 100</i>
